系統變數

:: 觀察作業系統環境變數
set
:: 將作業系統環境變數輸出至variables.txt
set > variables.txt

設定變數

基本設定

set LogName=myLog.txt

注意事項如下：

  1. 注意空白
  2. case insensitive

輸入提示

加上/p

運算式

加上/a

取得變數

前後加上%

ECHO %LogName%

常用變數

%CD% :當前工作目錄
%DATE% :目前系統日期
%TIME% :目前系統時間
%SystemDrive% :系統碟
%ProgramFiles% :應用程式目錄
%ProgramFiles(x86)% :應用程式目錄(x86)
